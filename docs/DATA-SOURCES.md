# Fontes de dados estratégicas, Casa do Licitante

## Objetivo
Consolidar bases públicas e semipúblicas úteis para inteligência comercial, análise institucional, qualificação de oportunidades, contexto territorial, capacidade fiscal e apoio à decisão para licitações.

## Fontes já reconhecidas como estratégicas

### 1. Licitações e contratações
- PNCP
- ComprasNet / Compras.gov / Dados Abertos de Contratações
- Portal da Transparência APIs

### 2. Capacidade fiscal, institucional e governamental
- CAPAG (Capacidade de Pagamento)
- Portal da Transparência
- Tesouro / finanças públicas relacionadas

### 3. Contexto territorial e estatístico
- IBGE
- bases municipais e territoriais públicas
- indicadores regionais e demográficos

### 4. Empresas, fiscalização e diligência
- CNPJ / CNAE
- dados cadastrais públicos correlatos
- CEIS / CNEP / sanções
- terceiro setor, entidades e organizações relacionadas quando úteis para diligência e contexto institucional

## Uso estratégico no produto
Essas bases devem servir para:
- enriquecer licitações
- contextualizar órgãos e municípios
- medir risco e atratividade
- apoiar análise comercial
- qualificar fornecedores e empresas
- apoiar leitura territorial e institucional
- reforçar triagem, jurídico e operação

## Diretriz de organização
Organizar cada fonte com estes campos:
- nome
- categoria
- URL de documentação
- autenticação
- cobertura
- dados principais disponíveis
- casos de uso no produto
- prioridade
- status de integração
- observações

## Backlog inicial de catalogação
1. Portal da Transparência API
2. CAPAG
3. IBGE
4. bases de terceiro setor relevantes
5. fontes fiscais/municipais/territoriais complementares

## Regra operacional
Toda fonte nova deve ser avaliada não só por disponibilidade da API, mas por:
- valor real para decisão comercial
- confiabilidade dos dados
- possibilidade de enriquecer fluxo de licitação
- aderência ao posicionamento da Casa do Licitante como central operacional
