# Imagens de Referência

Capturas enviadas pelo usuário como referência de produto/UX para o Casa do Licitante.

## Objetivo
Estas imagens documentam padrões de referência observados em outra plataforma, especialmente em:
- busca avançada e filtros
- análise de mercado
- contratações futuras
- empresas e documentos
- peças jurídicas
- navegação por áreas operacionais

## Uso correto
- usar como referência estrutural e de experiência
- não copiar identidade visual de forma literal
- adaptar os aprendizados para a proposta própria do Casa do Licitante
- priorizar interatividade, clareza, profundidade operacional e leitura profissional

## Resumo do que foi absorvido
A referência reforça que o produto deve evoluir em quatro camadas:
1. oportunidades / busca
2. análise / inteligência de mercado
3. negócios / empresas / documentos
4. jurídico / disputa / operação
